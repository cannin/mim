Description of the Schema taken from the XSD file
The MIM Visual (MIM-Vis) XML schema. The schema represents components of MIM diagrams and provides additional components to support metadata related to MIM diagrams. This schema is based on the GPML 2008/2010 XSD schemas with modifications primarily for validation purposes.
