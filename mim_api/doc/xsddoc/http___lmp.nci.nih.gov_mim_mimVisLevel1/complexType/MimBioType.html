Documentation
Information about cross-references, comments dealing with biology, semantic information about the interactions should be stored in MIM Bio using the IDs as the linking mechanism. Content in this container unlike the Comment or mimVis:GenericProperty will be strictly validated. The first version will not reference a seperate MIM Bio document.
Properties
This component is not nillable.
